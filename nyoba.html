HTML bagian 1

HTML adalah kependekan dari Hyper text Markup Language, yaitu sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web. HTML inilah yang menyusun sebuah halaman web menjadi sebagaimana yang kita lihat melalui browser. So, belajar HTML dan mengerti bagaimana cara HTML bekerja akan sangat penting jika kamu ingin terjun ke dunia web development.

Sebuah file HTML merupakan fie teks biasa yang mengandung tag-tag HTML. karena merupakan teks, maka HTML dapat dibuat dengan menggunakan teks editor yang sederhana, misal notepad. dapat juga menggunakan HTML editor yang bersifatWSIWYG (What You See Is What You Get), misal frontpage atau notepad++.
